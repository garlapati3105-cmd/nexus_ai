# AI Command Center Operations Dashboard

This dashboard is the primary operations control center for Nexus AI. It maps the active LangGraph workflows, displays agent statuses live, aggregates business optimizations, and provides verifiably audit trails.

## Layout & Components

1. **Stateful Graph Visualizer**: Runs an SVG-based topological diagram showing node lines, circles, and halos highlighting the active executing agent.
2. **Execution Tracer**: Chronological execution logs detailing downstream mutations and steps.
3. **Active Fleet Telemetry**: Displays health levels, latencies, confidence metrics, and utilized tools for every agent.
4. **Audit and Explainability Engine**: Shows reasoning outputs, alternatives evaluated, risk levels, and outcomes.
5. **RAG Knowledge Retrieval**: Shows documents accessed, similarity coefficients, and extracted chunks.
